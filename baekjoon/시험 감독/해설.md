- 해설
```python
N = int(input())
applicant = list(map(int, input().split()))
B, C = map(int, input().split())
answer = 0

for num in applicant:
    if num >= B: 
        num -= B # 총감독관은 B명
        if num % C == 0: # 남은 응시자의 수에서 부감독관(C)를 나눈다.
            answer += num // C
        else:
            answer += num // C + 1
            
print(answer + N) # 총감독관은 무조건 배치 된다.
```
